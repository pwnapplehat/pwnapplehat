## About

No CS degree. No bootcamp. I build production software end-to-end inside **Cursor** — AI-assisted development, not vibe coding. The AI writes fast; I stay accountable for every line: I read the diffs, question output that looks off, and verify behavior before anything reaches users. I never ship what I can't vouch for.

Solo, self-taught, and constantly learning. Based in India.
